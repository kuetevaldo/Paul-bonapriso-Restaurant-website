# PAUL Bonapriso Restaurant Website

A modern and responsive restaurant website designed for **PAUL Bonapriso**, a French bakery and restaurant located in Douala, Cameroon.

This project showcases a premium restaurant web experience with an elegant interface, smooth animations, multilingual support, and an integrated reservation system.

## Features

- Responsive design for mobile, tablet, and desktop
- Online table reservation system (EmailJS)
- English / French language switch
- Interactive menu filtering
- Restaurant gallery with lightbox
- Floating WhatsApp contact button
- Smooth animations and modern UI
- Google Maps location integration

## Technologies Used

- HTML5
- CSS3
- JavaScript
- EmailJS
- Font Awesome
- AOS Animations
- Lightbox Gallery

## Purpose

This project was developed as a **portfolio web design project** to demonstrate modern restaurant website development and user experience design.
